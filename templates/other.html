• NASHIK PLOGGERS WEBSITE

View Project

This website was created using the MEVN (MongoDB, Express, Vue.js, Node.js) stack. It is a platform for the Nashik Ploggers community to showcase their events, initiatives, and volunteering opportunities. The website features a responsive design, user authentication, and integration with Google Maps API to show the location of events. The MEVN stack was chosen for its ease of use, scalability, and flexibility.
